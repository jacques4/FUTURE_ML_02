# 🎫 Support Ticket Classification & Prioritization System

## Business Problem
Support teams receive thousands of tickets daily. Manual sorting delays urgent issues and wastes time.

## Solution
An ML-powered NLP system that:
- Automatically categorizes tickets
- Assigns priority levels
- Improves response speed and customer satisfaction

## Tech Stack
Python, Pandas, Scikit-learn, NLP (TF-IDF), Matplotlib

## Models
- Logistic Regression for classification
- TF-IDF text vectorization

## Results
- High accuracy on ticket categories
- Clear priority prediction
- Business-ready automation

## Business Impact
✔ Faster issue resolution  
✔ Reduced backlog  
✔ Smarter resource allocation
