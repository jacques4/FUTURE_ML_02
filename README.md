# 🎫 Support Ticket Classification & Prioritization System  
Machine Learning & NLP Project – Future Interns (2026)

## 📌 Project Overview

Customer support teams receive hundreds or thousands of tickets every day.  
Manual ticket sorting leads to delays, missed urgent issues, and inefficient workflows.

This project builds an intelligent Machine Learning system that automatically:

- Classifies support tickets into categories  
- Predicts ticket priority levels  
- Helps businesses respond faster and smarter  

---

## 🎯 Business Objectives

✔ Automate ticket categorization  
✔ Detect urgent issues early  
✔ Reduce backlog  
✔ Improve customer satisfaction  

---

## 📊 Dataset

Real-world styled customer support ticket dataset containing:

- Ticket subject & description  
- Ticket type (category)  
- Ticket priority  

---

## ⚙️ Technologies Used

- Python  
- Pandas & NumPy  
- Scikit-learn  
- NLP (TF-IDF Vectorization)  
- Matplotlib & Seaborn  

---

## 🧠 Machine Learning Pipeline

### 1. Data Loading & Exploration  
Understanding ticket structure and labels.

### 2. Text Cleaning & Preprocessing  
- Lowercasing  
- Removing noise & special characters  
- Normalizing text  

### 3. Feature Extraction  
TF-IDF converts text into numerical vectors.

### 4. Classification Tasks

#### 🎫 Ticket Type Classification  
Predicts support category.

#### 🚨 Ticket Priority Prediction  
Predicts urgency level.

### 5. Model Evaluation  
- Accuracy  
- Precision  
- Recall  
- Confusion Matrices
